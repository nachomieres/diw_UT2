Modelo de cajas

Es una de las características más importantes en las hojas de estilos. Cuando se crea una página en HTML, el uso de cada etiqueta simula lo que podría considerarse una “caja”, dotando a cada una de ellas de un identificador, indicado por el nombre de la etiqueta. Este sistema, nos permite, desde el fichero CSS, seleccionar cada caja y dotarle del estilo deseado.

El uso de las guías de estilo con CSS permite modificar todas las características de cada una de las cajas creadas.

Las propiedades principales con las que trabaja el modelo de cajas son “height” y “width” que harán referencia a las dimensiones de la cada de contenido o content. En este caso, estamos hablando del alto y el ancho del área donde se muestra el contenido de la caja.

Los márgenes tienen un comportamiento un tanto peculiar llamado “margin collapsing”. Cuando dos cajas se tocan en un mismo diseño, la distancia entre ellas es el valor del margen más grande, y no la suma de ambas.

Herencia de estilos

Cuando hablamos de CSS, la herencia es el mecanismo mediante el cual determinadas propiedades de un elemento “raíz o padre” se transmiten a sus “hijos”. No todas las propiedades CSS son heredables.

Los márgenes no se heredan porque es poco probable que un elemento “hijo” necesite los mismos márgenes que su “padre”. Cuando se establece el valor de una propiedad CSS en un elemento, sus elementos descendientes heredan de forma automática el valor de esa propiedad.

Como ya hemos comentado, existen algunas propiedades que no son heredables de “padres a hijos” pero existe un mecanismo mediante el cual es posible indicar que un elemento herede el valor de una propiedad. Cierre de guias de estilo

CSS Grid layout es una tecnología para desarrolladores web que contiene funciones de diseño. El CSS grid se puede utilizar para lograr muchos diseños diferentes y destaca por permitir dividir una página en áreas o regiones principales, por definir la relación en términos de tamaño, posición y capas entre partes de control construidas a partir de HTML.

Al igual que ocurre con las tablas el Grid layout permite a un desarrollador colocar contenido en filas y columnas. Sin embargo, con CSS grid son posibles muchos más diseños atendiendo a una forma más sencilla de diseñar el sitio web que haciendo uso de tablas. Por ejemplo, los elementos secundarios de un contenedor de cuadrícula podrían posicionarse para que se solapen y se superpongan de forma similar a los elementos proporcionados en CSS.
